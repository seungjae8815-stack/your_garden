# Play Console 등록 키트 — 너의 정원

비공개 테스트 업로드 시 콘솔에 그대로 복붙/선택하면 되는 자료. (2026-06-25 기준)

## 0. 업로드 파일
- **AAB**: `your_garden/build/app/outputs/bundle/release/app-release.aab`
- **앱 아이콘(512)**: `store_assets/play_icon_512.png`
- **피처 그래픽(1024×500)**: `store_assets/play_feature_1024x500.png`
- **스크린샷**: `store_assets/screenshots/` (폰에서 캡처 — 최소 2장, 권장 4~8장)
- **개인정보처리방침 URL**: GitHub Pages 게시 후 입력 (예: https://<아이디>.github.io/<repo>/)

---

## 1. 기본 정보
- **앱 이름**: 너의 정원
- **카테고리**: 건강/피트니스 (또는 라이프스타일)
- **태그**: 셀프케어, 일기, 명상/마음챙김, 힐링
- **무료/유료**: 무료 (인앱구매 없음, 광고 없음)

## 2. 짧은 설명 (최대 80자)
```
마음 한 줄을 묻으면, 그 마음이 양분이 되어 식물이 자라는 조용한 셀프케어 앱.
```

## 3. 전체 설명 (최대 4000자)
```
하루를 살다 보면 누군가에게 털어놓기엔 애매하고, 그냥 삼키기엔 무거운 마음이 쌓입니다.

'너의 정원'은 그 마음을 흙에 묻는 곳입니다. 비우거나 버리는 게 아니라, 묻는 거예요.
당신이 적은 한 줄은 사라지지 않고 식물의 양분이 되어, 천천히 자라나는 식물로 돌아옵니다.

■ 이렇게 작동해요
· 오늘의 마음을 한 줄 적습니다.
· 그 마음이 흙에 스며들고, 당신의 식물이 한 뼘 자랍니다.
· 식물이 건네는 짧은 한마디가 당신을 토닥입니다.

■ 조용하고, 사적이고, 압박 없이
· 누구에게도 보이지 않는 나만의 공간이에요.
· 연속 기록을 강요하지 않아요. 쉬어도 괜찮아요.
· 따뜻한 손그림 느낌의 코지한 식물과 함께.

■ 안전하게
· 힘든 마음이 깊어 보일 때, 도움받을 수 있는 상담 연락처를 부드럽게 안내해요.
· 이 앱은 의료·상담 서비스가 아니며, 정보 제공과 자기돌봄을 돕기 위한 것이에요.

오늘 마음 한 줄, 당신의 정원에 묻어보세요.
```

## 4. 앱 콘텐츠(App content) 양식

### 4-1. 개인정보처리방침
- URL 입력: (GitHub Pages 게시 URL)

### 4-2. 데이터 보안 (Data safety) — 답안
- **데이터를 수집/공유하나요?** 예 (수집함, 공유 안 함)
- **수집 항목**:
  - *앱 활동 → 사용자 생성 콘텐츠(기타)*: 사용자가 작성한 글. 목적: **앱 기능**. (계정 삭제 시 삭제됨)
  - *식별자 → 사용자 ID*: 익명 계정 식별자 + 앱이 생성한 임의의 기기 식별값. 목적: **앱 기능**. (광고ID 아님)
- **제3자 공유**: 없음
- **전송 중 암호화**: 예 (HTTPS)
- **데이터 삭제 요청 방법 제공**: 예 (앱 데이터 삭제 / 문의 이메일)
- **수집 안 함**: 이름·이메일·전화번호·정확한 위치·연락처·사진·금융정보 등

### 4-3. 콘텐츠 등급(설문) — 답안 가이드
- 폭력/성적 콘텐츠/욕설/약물 묘사: **모두 없음**
- 사용자 간 상호작용·콘텐츠 공유: **없음** (현재 비공개 솔로, 타인에게 노출 안 됨)
- 도박/디지털 구매: 없음
- → 예상 등급: **전체이용가(3+)** 수준

### 4-4. 대상 연령(Target audience)
- **만 13세 이상** 권장 (아동 대상 앱 아님 → '아동 포함' 선택 금지)

### 4-5. 광고
- **광고 없음** 선택

### 4-6. 데이터 보안: 정부앱/금융 등 해당 없음

## 5. 비공개 테스트 트랙 설정
1. Play Console → 앱 만들기(언어 한국어, 앱/게임=앱, 무료)
2. 테스트 → **비공개 테스트** → 새 버전 만들기 → AAB 업로드
3. 테스터: 이메일 목록(또는 Google 그룹) 생성 → **20명 이상** 추가
4. **opt-in(참여) 링크** 복사 → 테스터에게 전달 (TikTok/Shorts 시청자 모집)
5. ⏱️ 신규 개인 계정은 **20명이 14일 연속 참여**해야 프로덕션 신청 가능 → 빨리 시작할수록 좋음
