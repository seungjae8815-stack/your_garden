## 6. 🌱 사용자 니즈 기반 기능 제안 (성장 로드맵)

기존 계획(타임랩스→온보딩→위젯→태그/인사이트)과 감정일기 앱 시장의 검증된 니즈를 합친 우선순위:

### P0 — 리텐션 직결 (출시 직후 1~2주기)
1. **홈 화면 위젯** — 오늘의 식물 + "마음 묻기" 바로가기. 감정일기 앱 리텐션의 최대 지렛대 (기록 마찰 제거 + 매일 눈에 보임). 이미 계획에 있음 — 최우선 유지.
2. **부드러운 연속 기록 ("정원을 돌본 날 N일")** — 죄책감 주는 스트릭이 아니라 "연속이 끊겨도 정원은 그대로"라는 톤. 놓친 날은 비워두되 복귀 시 "돌아와줘서 고마워요" 식물 답장. 리텐션 + 앱의 정서 유지.
3. **성장 타임랩스** — 씨앗→만개 과정을 15초 영상으로. 돌아보기 의식의 감동 증폭 + **공유 바이럴 소재**(TikTok/릴스에 최적화된 포맷 — 크리에이터 채널과 시너지).

### P1 — 애착·깊이 (1~2개월)
4. **월말 정원 편지** — 매월 말 식물이 보내는 한 달 요약 편지 (가장 많았던 감정, 자주 쓴 단어, 인상적이었던 날). 월 단위 돌아보기 의식 = 리텐션 훅 + 프리미엄 후보.
5. **기록 검색·태그 필터** — "그때 그 마음 다시 보기". 기록이 쌓일수록 필수가 되는 기능.
6. **식물 종 선택권** — 현재 첫 식물이 무작위. 온보딩에서 3종 중 고르게 하면 애착 형성 (＋나머지 종은 "키우다 보면 만나요"로 수집 동기).
7. **사진 한 장 첨부** — 글+기분에 사진 1장. 일기 앱 요청 1순위 기능. (Supabase Storage 사용)

### P2 — 확산 (3개월~)
8. **소셜 정원 v2** — 기존 기획(익명 정원 방문 + 감정표현 5종, 댓글 없음) 그대로. 1-8에서 닫아둔 공개 기능을 이때 제대로 열기.
9. **다국어(영어·일본어)** — 감정 표현 톤이 핵심이라 번역 품질 중요. 일본은 힐링/저널링 앱 수용도가 높은 시장.
10. **연말 Year in Pixels 캠페인** — 12월 "올해의 내 마음" 공유 시즌 이벤트. 연 1회 최대 바이럴 기회.

## 7. 💰 수익화 전략

원칙(기존 결정 유지): **광고 의존 ❌** (힐링 정서와 충돌), **핵심 감정 루프는 영원히 무료** (기록·성장·돌아보기·위기지원을 유료화하면 신뢰 붕괴).

### 추천 모델: 프리미엄 구독 "정원사 패스" (soft paywall)

| | 무료 | 정원사 패스 |
|---|---|---|
| 기록·성장·돌아보기·달력·알림·백업 | ✅ 전부 | ✅ |
| 식물 종 | 기본 3~4종 | 전체 + 시즌 한정종 |
| 정원 테마 (밤하늘·사계절·비 오는 정원) | 기본 1종 | 전체 |
| 타임랩스 내보내기 | 워터마크 | 고화질·무워터마크 |
| 인사이트 | 최근 14일 | 전체 기간 + 월말 편지 아카이브 |
| 앱 아이콘·잠금화면 꾸미기 | – | ✅ |

- **가격**: 월 4,400원 / **연 29,000원 (월 2,417원 꼴 — 연간 유도가 핵심)** / 평생권 79,000원(초기 한정). 감정일기 앱은 연간 결제 비중이 높음 — 연간 첫 결제 7일 무료 체험.
- **왜 이 구성인가**: 유료 경계선이 "기능"이 아니라 **"꾸미기·보관·심화"**라서 무료 사용자도 완전한 경험을 하고, 애착이 생긴 사용자가 자연스럽게 결제. 죄책감 없는 과금 = 앱 정서와 일치.
- **구현**: `in_app_purchase` 직접 구현보다 **RevenueCat**(무료 tier 월 $2.5k 수익까지) 권장 — 영수증 검증·복원·양 스토어 통합을 대신 처리. Supabase profiles에 entitlement 캐시.
- **수수료**: Google/Apple 모두 연 매출 100만 달러 이하 15%.

### 보조 수익 (구독과 병행)

1. **일회성 IAP**: 식물 종 팩(3,300원)·정원 장식 소품 — 구독 부담스러운 사용자용. 구독자는 전부 포함.
2. **연간 성장 책자** (기존 계획): 1년치 기록+식물을 PDF(9,900원)/실물 인쇄본(29,000원~)으로. 연말 Year in Pixels 캠페인과 묶으면 시즌 매출 + 최고의 감성 상품. 실물은 국내 소량 인쇄 API(북토리 등) 연동.
3. **후원(팁)**: "정원사에게 씨앗 선물하기" 일회성 소액 결제. 1인 개발 서사 + 크리에이터 팬층과 궁합이 좋음.

### 하지 말 것
- 전면·배너 광고 (정서 파괴, 감정일기 앱에서 이탈 1순위 원인)
- 기록 개수 제한·백업 유료화 (데이터 인질 = 리뷰 테러)
- 하드 페이월 온보딩 (설치 직후 결제창)

### 타임라인
1. **출시~2개월**: 100% 무료. 리텐션 데이터 확보 + 리뷰 쌓기 (수익화보다 별점 4.5+가 먼저).
2. **2~3개월**: 정원사 패스 출시 (기존 사용자에게 "초기 정원사 할인" 평생권 — 초기 팬 보상 + 초기 매출).
3. **연말**: Year in Pixels 캠페인 + 성장 책자.
4. **소셜 v2 이후**: 시즌 한정 식물·테마로 구독 가치 지속 보강.
